安装破解记录

  1. 环境准备：Linux、JDK8+、MySQL
  2. Confluence安装包：https://www.atlassian.com/software/confluence/download-archives
  3. 在服务器上wget下载链接完成后给文件执行权限chmod +x atlassian-confluence-x.x.x-x64.bin
  4. ./atlassian-confluence-x.x.x-x64.bin 根据提示一路回车和确认默认安装
  5. 安装完成后访问http://IP:8090 确认安装成功
  6. 破解准备：https://gitee.com/pengzhile/atlassian-agent 下载wget release到服务器中解压。
  7. 到服务器刚刚解压运行这个破解jar查看使用方法 java -jar atlassian-agent.jar -h
  8. 他返回了export JAVA_OPTS="-javaagent:..."复制好备用。
  9. 根据官方文档 确认到/etc/init.d/confluence 是运行文件 vim查看启动脚本。
  10. 确认启动路径为：/opt/atlassian/confluence/bin/start-confluence.sh 后使用vim编辑
  11. find export 在上面添加刚刚复制好的 export JAVA_OPTS="-javaagent:/home/atlassian-agent-v1.3.1/atlassian-agent.jar ${JAVA_OPTS}"
  12. sudo /etc/init.d/confluence restart重启Confluence
  13. 输入ps aux|grep agent 查看confluence java进程，确认被载入atlassian-agent.jar
  14. 复制网站中的服务器ID准备生成Confluence激活码，到服务器中atlassian-agent.jar的目录
  15. java -jar atlassian-agent.jar -p conf -m admin@iacg.moe -n Ghost -o Ghost -s xxxx-LASW-X5SS-CID5
  16. 输入上面生成的激活码到网站激活根据提示和自身需求下一步。
  17. 到数据库步骤如果选择MySQL需要根据提示到官网下载驱动
  18. 到驱动下载页面选择平台无关解压出jar放入confluence的lib路径（我是下载MySQL8.0）
  19. sudo /etc/init.d/confluence restart 成功后刷新页面看到MySQL选项可以输入
  20. 在创建数据库名为confluence的数据库字符集用utf8mb4_bin
  21. 在mysql链接参数中加入jdbc:mysql://localhost/confluence?sessionVariables=tx_isolation='READ-COMMITTED’
  22. 下一步就是根据自身需求去配置了，配置全部完成后准备ngnix代理下。
  23. Confluence安装完成，剩下的就是对空间和网站的配置。

破解插件

  1. 去查找新应用寻找插件、安装完成后到管理应用里去复制你刚下载应用密钥
  2. 进入atlassian-agent.jar 的目录下把你刚刚复制的应用密钥粘贴到下面-p参数后
  3. java -jar atlassian-agent.jar -p com.mxgraph.confluence.plugins.diagramly -m admin@iacg.moe -n Ghost -o AnHuiWater -s BM78-LASW-X5SS-CID5
  4. 把生成的激活码黏贴到应用密钥下下面的许可证关键字的框框里完成激活。
